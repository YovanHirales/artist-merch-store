# Mack Johnson — Artist Website

Official website for rap/hip-hop artist Mack Johnson. Live at [mackraps.com](https://www.mackraps.com).

## Features

- **Music Page** — Album cover and spotify / apple music profile links
- **Merch Store** — Product listings with size selection and image carousel
- **Checkout** — Stripe-powered payment integration with webhook handling

## Tech Stack

- [Astro](https://astro.build/) — Static site framework
- [React](https://react.dev/) — Interactive UI components (cart, checkout)
- [Tailwind CSS](https://tailwindcss.com/) + [DaisyUI](https://daisyui.com/) — Styling
- [Stripe](https://stripe.com/) — Payment processing
- [Vercel](https://vercel.com/) — Hosting and deployment

## Getting Started

```bash
npm install
npm run dev
```
